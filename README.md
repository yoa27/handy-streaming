# Netflix Clone

Prototype de site de streaming utilisant l'API TMDB.

## Fonctionnalités
- **Page d'Accueil :** Bannière dynamique et rangées de films par catégories.
- **Lecture de Bandes-annonces :** Cliquez sur un film pour voir sa bande-annonce YouTube.
- **Recherche :** Recherchez des films via la barre de recherche dans la navigation.
- **Ma Liste :** Ajoutez des films à vos favoris (sauvegardés localement).

## Stack Technique
- React + Vite + TypeScript
- Axios (API TMDB)
- Lucide React (Icônes)
- React YouTube & Movie Trailer (Vidéos)
- Vanilla CSS

## Installation
1. Clonez le dépôt.
2. `npm install`
3. `npm run dev`

L'application sera disponible sur `http://localhost:5173`.
